# Python 單字測驗

這是一個基於 Python 開發的英文單字測驗程式，具備自動紀錄學習熟練度與動態篩選弱點單字的功能。

## 核心功能
- **熟練度分析**：根據答題狀況調整單字熟練度，並優先測驗不熟悉的單字。
- **多種測驗模式**：支援英翻中、中翻英及混合模式。
- **選擇題**：自動從單字庫生成隨機干擾選項。
- **資料持久化**：測驗結果會自動存回 `word.txt`。

## 如何使用
1. 確保電腦已安裝 Python 3。
2. 準備 `word.txt`，格式為：`英文 中文 熟練度` (例如：`apple 蘋果 0`)。
3. 執行 `main.py`：
   ```bash
   python main.py
